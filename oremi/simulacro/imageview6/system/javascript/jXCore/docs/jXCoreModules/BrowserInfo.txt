How do I use this module?
------------------------------
jXCore.include('BrowserInfo', './PathTojXCore');

What Does it do?
------------------------------
Will tell you more about the browser jXCore is running in.
e.g.

var BInfo = jXCore.BrowserInfo;
alert(BInfo.userAgent + ' ' + BInfo.version);
